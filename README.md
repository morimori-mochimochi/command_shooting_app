# ◾️ サービス概要
Docker、Linux、Gitコマンドを早打ちしてタイムを競うアプリです。<br>
ログインすると上位30位までが表示されるランキングにも参加可能ですので、<br>
打って打って打ちまくってランキング上位を目指しましょう！

[アプリはこちら](https://command-shooting-app-1.onrender.com)
# ◾️ 開発に至った経緯
カリキュラムに取り組む中で些細なものだけど積もれば膨大な時間となる、タイピング。<br>
私自身も、打ち慣れていない分キーに思い通りに指が運べずにたくさんタイポをしましたし、Mattermostでも多くの方がタイポに苦しめられたとの振り返りをされていました。<br>
「よく使うコマンドだけでもかっこよく、タタターン！と打てたら気持ちいいだろうなぁ」<br>
そんな思いからこのアプリの制作に至りました。

# ◾️ ユーザー層について
* プログラミング始めたばかりでコマンドに慣れない！
* 使い慣れてないからしょっちゅうタイポしちゃう...
* よく使うコマンドくらいはスラスラ打てるようにしたい！<br>
そんなビギナーの皆さんのためのタイピング練習アプリです。

# ◾️ サービス利用方法
1. 「ログイン」ボタンをクリックし、次に表示される「新規登録」からユーザー登録を行う。<br>
(ログインなしでも利用可能ですが、プレイ結果は破棄されます。）
2. 「早打ちする」ボタンをクリックし、「スタートボタン」を押すとゲームが始まります。
3. ゲームが終了するとタイムが表示されます。
4. 「ランキング」ボタンを押すと１〜30位までの方の結果が表示されます。
